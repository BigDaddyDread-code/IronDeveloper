# Documentation Archive

**Status:** Canonical directory boundary

This directory is reserved for non-current documents whose live and generated references have been proven safe to update.

Archiving changes navigation, not history. Archived material remains inspectable and cannot become current authority. Receipts stay under `Docs/receipts/`; they are never moved here merely because they are old.

No legacy document was moved during CLN-05 because automated repository-wide link and contradiction enforcement is not available until CLN-07.
