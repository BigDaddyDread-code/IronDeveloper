# Slow Test / Quarantine Register

Quarantine is not deletion.

A slow test in a new bucket is still a slow test.

Selection proof is not execution proof.

Selection proof means a filter lists tests.

Execution proof means the selected tests ran and passed.

This PR does not treat selection proof as execution proof.

## Owner Debt

Rows using `owner-required` need a real owner in a later test ownership pass. The value is explicit debt, not an empty owner.

## Category Status

- Slow: not introduced in G14.
- LongRunning: introduced for real database and store-shaped integration classes.
- Quarantined: not introduced in G14.
- RequiresRealDatabase: introduced for real database and store-shaped integration classes.
- RequiresExternalDependency: not introduced in G14.
- RequiresLocalTooling: not introduced in G14.
- ManualLocal: introduced for the existing ignored manual local indexing task.
